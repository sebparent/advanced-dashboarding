Lance l'agent `wrap-up` (voir `.claude/agents/wrap-up.md`) pour produire une synthèse
dense et fidèle de l'état actuel du projet : architecture, cloisonnement client,
modèle de données Metabase, fonctionnalités, état d'avancement, et prochaines actions.

Utilise l'outil Agent avec `subagent_type: wrap-up`.
